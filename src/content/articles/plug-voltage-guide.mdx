# 各國插頭電壓懶人包：一篇搞定轉接頭

出國旅遊最怕到了飯店才發現插頭插不進去，或是電器接上電源後直接燒壞。全球插頭規格超過 15 種，電壓也分成兩大陣營。這篇幫你一次搞懂，出發前就備好正確的轉接頭。

## 四大常見插頭類型

### Type A / Type B（台灣、日本、美國）

台灣最熟悉的兩腳扁插與三腳扁插。日本同樣使用 Type A，但要注意日本的插座孔位較窄，如果你的插頭兩腳一寬一窄，可能會插不進去。美國則普遍使用帶接地的 Type B。

### Type C / Type F（歐洲、韓國）

歐洲大部分國家使用圓形雙腳插頭。Type C 是基本的兩圓腳，Type F 多了接地夾片。韓國也屬於這個系統，去首爾帶歐規轉接頭就對了。

### Type G（英國、新加坡、馬來西亞、香港）

三根方腳的大型插頭，體積是所有插頭中最大的。去英國、新加坡、馬來西亞、香港都需要這種轉接頭，務必提前準備。

### Type I（澳洲、紐西蘭）

兩根斜腳加一根接地腳，造型像倒 V。澳洲與紐西蘭專用，其他國家極少見，千萬別忘了帶。

## 電壓差異：110V vs 220V

| 電壓 | 使用國家 |
|------|----------|
| 110V | 台灣、日本、美國、加拿大 |
| 220-240V | 歐洲、英國、澳洲、韓國、中國、東南亞大部分國家 |

### 哪些電器不用擔心？

現代的**手機充電器、筆電充電器、相機充電器**幾乎都是寬電壓設計（100-240V），看一下充電器上的標示就知道。這類電器只需要轉接頭就能正常使用。

### 哪些電器一定要注意？

**吹風機、電棒捲、電熱水壺**通常是固定電壓。把 110V 的吹風機接上 220V 電源，輕則燒保險絲，重則冒煙起火。如果你非得帶這類電器，需要購買降壓變壓器。

## 實用建議

- 出發前用[插頭電壓查詢工具](/tools/plug-voltage)確認目的地規格
- 買一顆**萬國轉接頭**最省事，涵蓋 150 國以上的插座規格
- 旅館通常有吹風機，不需要自己帶出國冒風險
- 搭配[行李打包清單](/tools/packing-list)確認轉接頭已放入行李

搞定插頭電壓問題，旅途中就不用擔心手機沒電或相機罷工的窘境。
