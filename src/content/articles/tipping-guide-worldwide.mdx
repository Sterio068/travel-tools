# 各國小費文化完整指南：該給多少？怎麼給？

小費文化因國家而異，給錯金額可能讓人覺得失禮，不該給的地方硬給反而尷尬。這篇整理台灣旅客最常去的國家，幫你搞懂哪裡該給、給多少、怎麼給。

## 不需要給小費的國家

### 日本

日本是**完全不需要給小費**的國家。給小費反而可能讓對方感到困惑甚至被冒犯，因為日本文化認為提供優質服務是理所當然的。旅館、餐廳、計程車都不用給，請安心收起你的零錢。

### 韓國

韓國沒有小費文化，餐廳不需要額外給小費。部分高級餐廳帳單已含 10% 服務費，不需要再另外加。

### 中國

中國大陸一般不需要給小費。跟團旅遊時導遊和司機可能期望小費，每人每天約 **人民幣 20-50 元**是常見行情。

## 建議給小費的國家

### 美國（幾乎是強制）

美國的小費文化根深蒂固，不給小費等同於表達不滿。

| 場合 | 小費標準 |
|------|----------|
| 餐廳 | 帳單金額的 **15-20%**（午餐 15%、晚餐 18-20%） |
| 酒吧 | 每杯飲料 **1-2 美元** |
| 計程車 / Uber | 車資的 **15-20%** |
| 飯店行李員 | 每件行李 **1-2 美元** |
| 客房清潔 | 每晚 **2-5 美元** |

### 泰國（隨意但受歡迎）

泰國的小費文化輕鬆隨意，不給不會被白眼，但給了會讓對方很開心。

- **餐廳**：找零不取回即可，或留 **20-50 泰銖**
- **按摩**：**50-100 泰銖**，依滿意程度調整
- **飯店行李員**：**20 泰銖**
- **計程車**：通常不給，湊整數即可

### 歐洲（適度給予）

歐洲多數國家帳單已含服務費，小費屬於額外感謝。

- **餐廳**：帳單**抹零或加 5-10%**。例如帳單 47 歐元，留 50 歐元
- **計程車**：湊到整數即可
- **飯店**：行李員每件 **1-2 歐元**

### 東南亞其他國家

- **越南**：餐廳不強制，高級餐廳留 **5-10%**
- **印尼 / 峇里島**：餐廳留 **10,000-20,000 印尼盾**
- **菲律賓**：餐廳 **10%**，部分已含服務費

## 給小費的通用技巧

- 永遠用**當地貨幣**給小費，不要給台幣或美金硬幣
- 出發前用[匯率換算工具](/tools/currency)算好當地幣值的小費金額
- 準備足夠的**小面額鈔票**，避免臨時找不開
- 搭配[各國旅遊須知](/tools/country-info)查詢目的地的小費習慣

掌握各國小費規則，不只展現國際禮儀，也讓旅途中的每次互動更加順暢。
