# Visit Japan Web 2026 最新教學

Visit Japan Web（VJW）是日本政府推出的入境線上服務，整合了入境審查、海關申報和免稅購物等功能。提前在 VJW 完成登錄，入境時只要出示 QR Code 就能快速通關，省去手寫紙本表格的麻煩。

## 什麼是 Visit Japan Web？

VJW 是日本數位廳開發的網頁服務（不是 App），旅客可以提前在線上填寫入境資料，生成 QR Code 供入境時使用。目前涵蓋三大功能：

- **入境審查**：取代過去的紙本入境卡
- **海關申報**：取代紙本海關申告書
- **免稅購物登錄**：店家免稅資料直接綁定護照

## 註冊與登錄步驟

### 步驟一：建立帳號

前往 [Visit Japan Web 官網](https://www.vjw.digital.go.jp/) 註冊帳號，需要準備：

- 有效的電子信箱
- 自訂密碼（至少 10 碼，含大小寫英文和數字）

### 步驟二：登錄個人資料

帳號建立後，填寫基本資料：

- 護照上的英文姓名（務必與護照完全一致）
- 出生日期
- 國籍
- 護照號碼與有效期限
- 護照照片頁拍照上傳

### 步驟三：登錄旅程資訊

新增一筆旅行計畫，填寫：

- 抵達日本的航班編號與日期
- 在日本的住宿地址（飯店名稱和地址）
- 預計停留天數
- 入境目的（觀光）

### 步驟四：填寫入境審查與海關申報

- **入境審查**：確認個人資料無誤後送出
- **海關申報**：如實申報攜帶物品，包含是否攜帶超額現金、菸酒等

### 步驟五：取得 QR Code

資料送出後系統會生成兩組 QR Code：

1. **入境審查用 QR Code**（藍色畫面）
2. **海關申報用 QR Code**（黃色畫面）

建議提前截圖保存，避免在機場網路不穩時無法讀取。

## QR Code 使用時機

抵達日本機場後，依序在以下關卡出示 QR Code：

1. **入境審查**：在自動通關閘門或人工審查窗口掃描藍色 QR Code
2. **領取行李**後
3. **海關出口**：掃描黃色 QR Code 通過海關

## 常見問題與錯誤排除

### 姓名順序填錯

VJW 的姓名欄位是「Surname（姓）」在前、「Given name（名）」在後，和部分訂機票平台相反，務必對照護照填寫。

### 照片上傳失敗

護照照片頁拍照時注意光線充足、避免反光，檔案大小不要超過限制。建議用手機拍攝後直接上傳。

### QR Code 無法顯示

確認瀏覽器已登入 VJW 帳號，且旅程狀態顯示為「登錄完成」。如果仍有問題，清除瀏覽器快取後重新整理。

## 實用小提醒

- 出發前 **兩週** 即可開始填寫，不必等到出發當天
- 同行旅伴需要各自建立帳號，但一個帳號可以幫家人（如小孩）登錄
- VJW 僅支援網頁版，任何瀏覽器都可以使用
- 搭配[出國行李清單](/articles/packing-checklist-ultimate)一起準備，確保出發前萬無一失
